OGFImg Project

General

OGFImg is a project I made (that is not being updated by nouxinf) to download then "stitch" tiles from any tile map of your choosing.

Right now it has 7 layers to download from :D

Installation

To get started with OGFImg, simply download the latest version from the GitHub repo Here.

Download all of the dependencies with:

pip install flask requests pillow

Usage

After installation, you can run the following command to start using OGFImg:

python app.py

(or just run the app.py)

From there move to localhost:5000

Fun Facts

When developing OGFImg, I experimented with several Python libraries to handle image processing efficiently, including Pillow and OpenCV. It was a fun journey!
